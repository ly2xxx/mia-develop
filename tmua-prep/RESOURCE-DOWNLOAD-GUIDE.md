## üì• Download Instructions

### 1. TMUA Official Materials
**Website:** https://esat-tmua.ac.uk/prepare/

**Steps:**
1. Go to https://esat-tmua.ac.uk/tmua-preparation-materials/
2. Download:
   - TMUA Test Specification PDF
   - Logic and Proof notes
   - All available past papers (2018-2025)
   - Sample papers and worked solutions
3. Save to: `C:\code\tmua-prep\past-papers\TMUA-Official\`

**File naming convention:**
```
TMUA_2024_Paper1.pdf
TMUA_2024_Paper2.pdf
TMUA_2024_Solutions.pdf
```

---

### 2. MAT Past Papers
**Website:** https://www.maths.ox.ac.uk/study-here/undergraduate-study/maths-admissions-test

**Steps:**
1. Navigate to past papers section
2. Download papers from 2015-2024 (MAT Section A focus)
3. Download mark schemes and feedback reports
4. Save to: `C:\code\tmua-prep\past-papers\MAT\`

**Note:** MAT is being replaced by TMUA for Oxford from 2026, but historic papers are excellent practice

**File naming:**
```
MAT_2024_Paper.pdf
MAT_2024_Solutions.pdf
MAT_2024_Feedback.pdf
```

---

### 3. STEP Foundation Modules
**Website:** https://maths.org/step/

**Steps:**
1. Go to STEP Support Programme website
2. Navigate to "Assignments" or "Foundation Modules"
3. Download modules 1-4 (Foundation level)
4. Download corresponding solutions
5. Save to: `C:\code\tmua-prep\resources\STEP-Foundation\`

**Alternative source:** https://stepdatabase.maths.org/

**File naming:**
```
STEP_Foundation_Module1.pdf
STEP_Foundation_Module1_Solutions.pdf
```

---

### 4. PAT Past Papers (Maths Only)
**Website:** https://www.ox.ac.uk/admissions/undergraduate/applying-to-oxford/guide/admissions-tests/pat

**Steps:**
1. Download PAT past papers (2015-2024)
2. Extract only mathematics sections (ignore physics)
3. Download mark schemes
4. Save to: `C:\code\tmua-prep\past-papers\PAT\`

**Note:** PAT Section 1 contains maths questions useful for TMUA practice

---

### 5. ENGAA & ECAA Historic Papers
**Website:** https://esat-tmua.ac.uk/esat-preparation-materials/ (historic papers section)

**Steps:**
1. Download historic ENGAA papers (pre-2026, now replaced by ESAT)
2. Download historic ECAA papers
3. Focus on mathematics sections only
4. Save to: `C:\code\tmua-prep\past-papers\ENGAA\` and `\ECAA\`

---

### 6. AMC 12 Papers
**Website:** https://www.maa.org/math-competitions/amc-1012

**Steps:**
1. Navigate to "AMC 12 Archive"
2. Download last 5 years of papers (2020-2024)
3. Download answer keys
4. Save to: `C:\code\tmua-prep\practice-tests\AMC12\`

**Strategy:** Only practice first 15 questions per paper (easier/medium difficulty)

**Note:** AMC papers available for free after competition season ends

---

### 7. UKMT Resources (Optional)
**Website:** https://www.ukmt.org.uk/competitions/solo-competitions

**Steps:**
1. Navigate to "Past Papers" section
2. Download Senior Mathematical Challenge papers
3. Download Kangaroo (follow-on round) papers
4. Save to: `C:\code\tmua-prep\practice-tests\UKMT\`

**Note:** Available for registered teachers/previous participants; check availability

---

### 8. Anki Setup
**Website:** https://apps.ankiweb.net/

**Steps:**
1. Download Anki for Windows/Mac/Linux
2. Install and launch
3. Create new deck: "TMUA 2026"
4. Create sub-decks:
   - TMUA 2026::Paper 1 - Applications
   - TMUA 2026::Paper 2 - Reasoning
   - TMUA 2026::Common Mistakes
   - TMUA 2026::Quick Techniques
5. Optional: Download AnkiWeb account for cloud sync

**First cards to create:**
- TMUA exam format (papers, timing, scoring)
- Key formula reminders
- Logic symbols and proof techniques

## üìä Organization Tips

### Folder Naming
Use consistent, sortable names:
```
YEAR_EXAM_Paper_Type.pdf

Examples:
2024_TMUA_P1_Questions.pdf
2024_TMUA_P1_Solutions.pdf
2023_MAT_P1_Questions.pdf
```

### Master Spreadsheet
Create `test-scores.xlsx` with columns:
- Date
- Resource (e.g., "MAT 2023 P1")
- Questions attempted
- Correct answers
- Time taken
- Accuracy %
- Topics weak on
- Notes

### Cloud Backup
**Recommended:** Back up to OneDrive/Google Drive
- Folder: `TMUA Prep 2026`
- Enable auto-sync
- Access from multiple devices

## üÜò Troubleshooting

**Problem:** Can't find official TMUA past papers  
**Solution:** Check esat-tmua.ac.uk or email UAT-UK directly

**Problem:** MAT website moved/changed  
**Solution:** Search "University of Oxford MAT past papers" or check Wayback Machine

**Problem:** STEP resources require login  
**Solution:** Registration is free for students; use school email if available

**Problem:** AMC papers want payment  
**Solution:** Older papers (3+ years) usually free; check MAA website carefully
