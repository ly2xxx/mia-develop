## üì• Download Instructions

### Method 1: Manual Download
1. Right-click each link above
2. "Save Link As..."
3. Save to appropriate folder in `past-papers/`

### Method 2: Batch Download (PowerShell)

Create `download-all-papers.ps1`:
```powershell
# TMUA Papers 2018-2024
$baseUrl = "https://esat-tmua.ac.uk/sites/default/files"
$years = 2018..2024

foreach ($year in $years) {
    Invoke-WebRequest "$baseUrl/$year-10/TMUA_${year}_Paper1.pdf" -OutFile "TMUA-Official/TMUA_${year}_Paper1.pdf"
    Invoke-WebRequest "$baseUrl/$year-10/TMUA_${year}_Paper2.pdf" -OutFile "TMUA-Official/TMUA_${year}_Paper2.pdf"
    Invoke-WebRequest "$baseUrl/$year-10/TMUA_${year}_Solutions.pdf" -OutFile "TMUA-Official/TMUA_${year}_Solutions.pdf"
    Start-Sleep -Seconds 1
}

# MAT Papers 2015-2024
$matUrl = "https://www.maths.ox.ac.uk/system/files/attachments"
$matYears = 2015..2024

foreach ($year in $matYears) {
    Invoke-WebRequest "$matUrl/${year}_MAT_paper.pdf" -OutFile "MAT/${year}_MAT_paper.pdf"
    Invoke-WebRequest "$matUrl/${year}_MAT_solutions.pdf" -OutFile "MAT/${year}_MAT_solutions.pdf"
    Start-Sleep -Seconds 1
}

Write-Host "Download complete!"
```

Run: `.\download-all-papers.ps1` from `past-papers/` directory

### Method 3: curl (if preferred)

# Download single TMUA paper
