## ðŸŽ“ Study Tips from 9.0 Scorer

### Mental Approach
1. **Understand > Memorize:** But use Anki to reinforce understanding
2. **Speed matters:** You need autopilot responses to familiar patterns
3. **70% recognition target:** If you've seen it before, you should solve it quickly
4. **Later questions harder:** Save time on early Qs to tackle difficult ones

### Exam Technique
1. **No negative marking:** Guess if stuck (but educated guess preferred)
2. **Time per question:** ~3.75 minutes average
3. **Flag and return:** Don't get stuck on one question
4. **Check arithmetic:** Silly mistakes cost marks

### Common Pitfalls
- Rushing through "easy" questions and making careless errors
- Spending too long on one hard question
- Not attempting all questions
- Forgetting to use all answer options (elimination strategy)
